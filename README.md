# Радар Мосбиржи — PWA для анализа российского рынка акций

Бесплатное PWA-приложение: живые котировки Мосбиржи, RSI, дивидендный календарь, ОФЗ, избранное, офлайн-режим.

## Данные

Котировки поступают с **открытого API Московской биржи (ISS MOEX)** — бесплатно, без ключей, без регистрации:
- Акции: `iss.moex.com/iss/engines/stock/markets/shares/boards/TQBR/securities.json`
- Индекс IMOEX: `.../markets/index/boards/SNDX/securities/IMOEX.json`
- ОФЗ: `.../markets/bonds/boards/TQOB/securities.json`
- Свечи для RSI: `.../securities/{TICKER}/candles.json?interval=24`

Задержка данных ~15 минут (стандарт для бесплатного доступа).

## Деплой на GitHub Pages (как vino-schedule)

```bash
# 1. Создай репозиторий, например moex-radar
git init
git add .
git commit -m "MOEX Radar PWA v1"
git branch -M main
git remote add origin https://github.com/KazemJ-ui/moex-radar.git
git push -u origin main

# 2. В настройках репозитория: Settings → Pages → Source: main branch / root
# 3. Приложение будет доступно на https://kazemj-ui.github.io/moex-radar/
```

ВАЖНО: PWA (service worker, установка) работает только по HTTPS — GitHub Pages подходит идеально.

## Структура

```
index.html     — разметка, app shell
styles.css     — тёмная терминальная тема
app.js         — логика: MOEX API, RSI(14), избранное, навигация
sw.js          — service worker: офлайн-кэш shell + network-first для API
manifest.json  — PWA-манифест (установка на телефон)
icon.svg/png   — иконки
```

## Возможности

- **Акции** — топ-60 по обороту, поиск, сортировка (оборот / рост / падение)
- **Детальная карточка** — свайп-шит с RSI(14), спарклайном за 40 дней, заметкой по эмитенту
- **Избранное** — звёздочки, хранится в localStorage устройства
- **Дивиденды** — календарь ближайших отсечек (статичный слой, обновляется в коде)
- **ОФЗ** — живые котировки гособлигаций с доходностью к погашению
- **Офлайн** — service worker отдаёт последние закэшированные данные
- **Установка** — баннер "Установить как приложение" (A2HS)

## Идеи для развития (v2)

- Push-уведомления при достижении целевой цены (нужен бэкенд — можно тот же сервер 87.236.23.84)
- Графики свечей (lightweight-charts от TradingView — бесплатная либа)
- Портфельный трекер: ввод своих позиций, расчёт P&L
- Сравнение акций бок о бок
- Скринер: фильтр по RSI < 30, дивдоходность > 10% и т.д.
- Монетизация: премиум-фичи (алерты, расширенный скринер) или реклама

## Лицензия данных

MOEX ISS — открытые данные с отложенными котировками. Для коммерческого
использования реального времени нужна лицензия Мосбиржи. Для бесплатного
приложения с 15-минутной задержкой — ок.
