О приложении

Радар Мосбиржи — бесплатный инструмент анализа российского рынка акций. Котировки поступают напрямую с Московской биржи через открытый API (ISS MOEX) с задержкой ~15 минут.

Что умеет

  • Живые котировки — цена, изменение, объём торгов по всем ликвидным акциям
  • RSI(14) — рассчитывается из дневных свечей за 3 месяца
  • Избранное — твой персональный список, хранится на устройстве
  • Дивидендный календарь — ближайшие отсечки
  • ОФЗ — котировки гособлигаций
  • Офлайн — приложение открывается без интернета (показывает последние данные)

Как читать RSI

<30 перепродано 30–70 нейтрально >70 перекуплено

⚠ Не является инвестиционной рекомендацией. Данные предоставляются «как есть». Перед сделками проверяй информацию в торговом терминале брокера.
